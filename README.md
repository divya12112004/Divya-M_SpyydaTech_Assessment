This repository contains all required programs for the SpyydaTech Technical Assessment.

---

##  Programs Included
| File Name | Description |
|----------|-------------|
| `word_frequency.py` | Counts frequency of unique words in a paragraph |
| `balanced_brackets.py` | Validates if brackets are balanced |
| `library_system.py` | Basic CRUD library management system |
| `lis_length.py` | Finds length of Longest Increasing Subsequence |
| `url_shortener.py` | Short URL generator with redirect functionality |
| `calculator_history.py` | Calculator with history support |

---

##  How to Run Code

Make sure Python is installed.

Open terminal inside project folder and run:
